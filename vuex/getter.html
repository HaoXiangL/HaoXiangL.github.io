# Getter

Getter属性相当于store的computed计算属性，在state返回到组件之前作出一些操作(如排序，过滤)，实现Getter的好处是复用性更好，并能根据依赖缓存起来。

store/index.js

state: {
  todos: [
    { id: 1, text: '可显示', done: true },
    { id: 2, text: '不可显示', done: false }
  ]
},
getters: {
  doneTodos: state => {
    return state.todos.filter(todo => todo.done)
  }
})

在组件中获取:

App.vue

export default {
computed: {
  todos(){
    return this.$store.getters.doneTodos;
  }
}

# 通过方法访问

你也可以通过让 getter 返回一个函数，来实现给 getter 传参。在你对 store 里的数组进行查询时非常有用。

getters: {
  // ...
  getTodoById: (state) => (id) => {
    return state.todos.find(todo => todo.id === id)
  }
}

# mapGetters 辅助函数

同样的，getters也有辅助函数，mapGetters 辅助函数仅仅是将 store 中的 getter 映射到局部计算属性：

import { mapGetters } from 'vuex'

export default {
// ...
computed: {
// 使用对象展开运算符将 getter 混入 computed 对象中
  ...mapGetters([
    'doneTodosCount',
    'anotherGetter',
    // ...
  ])
}
